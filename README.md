# Proyecto-HTML-y-CSS-desde-CERO

Objetivo del Proyecto
Debes diseñar una página de presentación para un restaurante (sin funcionalidades complejas, solo informativa) que sirva para mostrar en tu portfolio y prepararte para el examen final. Se permite el uso de plantillas o elementos de internet.

Secciones Requeridas
La web debe incluir las siguientes secciones:

Encabezado (Header): Debe tener el nombre del restaurante y un menú de navegación con enlaces a: Inicio, Menú, Nuestra Historia y Contacto.

Sección de Inicio: Debe incluir una descripción breve del restaurante y una imagen destacada. El enlace siempre debe redirigir a la página principal (index.html).

Sección de Menú: Una lista de los platos que ofrece el restaurante, incluyendo imágenes y descripciones. Puedes organizarlo usando tablas o el formato que prefieras.

Sección "Nuestra Historia": Un texto que cuente cómo surgió el negocio, quiénes son los fundadores y qué lo hace especial.

Sección de Contacto: * Un formulario de contacto (maquetado, aunque no tenga backend todavía).

Información del restaurante: dirección, redes sociales y número de teléfono.

Recomendaciones finales
Se sugiere buscar una plantilla de HTML y CSS atractiva para facilitar el diseño.

Es el paso previo al examen final, por lo que se recomienda ponerle esmero para que sume a tu perfil profesional.
